How to Use

1. Download the Project

Clone or download the project files.

git clone https://github.com/rahulkgamedev/Exam-Seat-Allocator

Or download the ZIP file and extract it.

2. Open the Project

Open the project folder and locate:

index.html

3. Run the Application
Option 1 — Direct Browser Run

Double-click the HTML file.

index.html

The application will open in your browser.

Option 2 — VS Code Live Server (Recommended)
Open project folder in VS Code
Install Live Server extension
Right click on:
index.html
Click:
Open with Live Server

4. Add Data

You can:

Add Students manually
Add Exams manually
Add Exam Halls manually

OR

Upload CSV files from the CSV Upload section.

5. Configure Halls

Add halls with:

Hall Name
Block
Floor
Room Type

Supported hall types:

Small Room
Standard Room
Large Hall
Auditorium
Custom Hall Range

6. Run Seat Allocation
Go to:
Run Allocation
Select the exam
Configure seating distance
Click:
Allocate All Students

7. View Results

After allocation:

Master seating table is generated
Hall-wise seating view is shown
Unallocated students are displayed (if capacity is insufficient)
8. Export Results

You can export:

Master Allocation CSV
Hall-wise CSV files

9. Features Included
Multi-hall allocation
Subject interleaving
Anti-cheating seating
Special needs prioritization
CSV upload/export
Responsive dashboard

10. Browser Compatibility

Recommended browsers:

Google Chrome
Microsoft Edge
Firefox
